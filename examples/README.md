# MCP Hub Examples

## Basic Examples

### Search for Servers
```bash
node basic/search-servers.js github
```

### Install a Server
```bash
mcphub install @official/github
```

## Advanced Examples

### Auto-Setup Development Environment
```bash
node advanced/auto-setup.js
```

This installs recommended servers for developers:
- GitHub integration
- Filesystem access
- Browser automation (Puppeteer)
